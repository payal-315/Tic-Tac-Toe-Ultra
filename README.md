# Tic-Tac-Toe-Ultra
A modern responsive Tic Tac Toe game built with Java Swing and custom particle effects.



A high performance Java Tic Tac Toe engine featuring a responsive dark mode UI and dynamic particle animations. Designed for a seamless cross platform desktop experience with integrated session score tracking.



CORE FEATURES
----------------
Responsive UI Automatically adapts to any screen size 80 percent height scaling with a fixed 7 10 portrait aspect ratio.

Floating Stickers A custom animation engine that renders semi transparent X and O particles drifting in the background void.

Blackout Victory A cinematic victory panel that overlays the board with high contrast symbols and bold typography.

Live Scoreboard Real time tracking of session wins for Player X and Player O.

Smooth Animations 60FPS double buffered rendering to ensure no UI flickering during gameplay.



INSTALLATION AND SETUP
-----------------------
To run this project locally follow these steps

1 Clone the Repo git clone https github com YOUR_USERNAME Tic Tac Toe Ultra git

2 Compile the Source Ensure you have the Java Development Kit JDK installed. javac tic_tac_toe java

3 Launch the Game java tic_tac_toe




HOW TO PLAY
-------------
Turn Based Action Player X starts first. Click any tile to place your mark.

Replay After a win or draw click REPLAY to start a new round while keeping the current score.

Reset All Click RESET ALL to wipe the scoreboard and start a completely fresh session.



TECH STACK
-----------
Language Java

Framework Swing AWT

Layout Managers GridBagLayout BorderLayout and GridLayout.

Graphics Graphics2D with Antialiasing for smooth particle effects.
